# Online Harassment Evidence Collection

A Windows Forms application built with **C# (.NET Framework)** to collect, manage, and organize online harassment evidence in a structured and secure way.  
The system uses **SQL Server Express** for case and evidence storage and is designed for academic and investigative purposes.

## üß© Database
- **Database Engine:** SQL Server Express
- **Backup File:** `login.bacpac`
- The database contains tables for users, cases, and evidence.

### Restore Database
1. Open **SQL Server Management Studio (SSMS)**
2. Right-click **Databases**
3. Select **Import Data-tier Application**
4. Choose `login.bacpac`
5. Finish the wizard
